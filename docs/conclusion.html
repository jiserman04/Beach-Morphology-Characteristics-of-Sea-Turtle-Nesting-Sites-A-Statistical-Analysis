Beach Morphology Characteristics of Sea Turtle Nesting Sites: A Statistical Analysis

Chapter 3 Conclusion

The goal for this project was to perform statistical analysis on the sea turtle nest locations and determine if there is a statistically significant difference between these nest locations and the others.

3.1 Summary: Interpretation

This analysis determined that statistically significant differences between the nest line and other locations exist. Foreshore slope differences exist between the nest line and 25E, 50E, 100E, and 100W, as well as between 50E and 50W. Beach Slope differences exist between the nest line and 25E, 25W, 50E, 50W, and 100W, as well as between 100W and 100E. Beach width differences exist between the nest line and 25W and 25E. No significant differences in foredune height were found between any variables. This analysis shows that foreshore slope and beach slope are important for nest site selection, while beach width and foredune height are relatively the same throughout the beach.

3.2 Limitations

There were limitations for the project that may have affected results. The data was collected only on Pensacola Beach, which may or may not apply to other locations. The data also consisted of fourteen loggerhead turtle nests and one green turtle nest, which is an uneven balance of species. Inference cannot be made on green turtles due to the lack of sample size and thus, was excluded from analysis. Fourteen is also a relatively small sample size. A larger sample size will provide more accurate estimation, tighter confidence intervals, and the ability to perform more complex statistical analysis, such as modeling using regression analysis. Finally, this data only examines sea turtle nesting behavior for a single year; longitudinal data should be collected for a more complete picture of sea turtle preferences.
